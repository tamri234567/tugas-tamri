Tag input dengan properti type="text"


Ini textbox dengan nama teks_ku ukuran 16 chr dan maksimal karakter yang dapat ditampung 20 chr(character).

Tag input dengan properti type="password"


Ini passwordbox dengan nama tpassword_ku ukuran 16 chr dan maksimal karakter yang dapat ditampung 20 chr(character).

Tag input dengan properti type="radio"

Radio 1 --> nama radio_ku isi/value isi_1
Radio 2 --> nama radio_ku isi/value isi_2
Jika nama radio tidak sama maka tidak dianggap satu pilihan. Untuk membuktikannya silahkan ganti salah satu nama dari radio diatas untuk mengetahui perbedaanya.
